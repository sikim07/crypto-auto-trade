### 전략 C — BB수축 + 상단돌파 + 거래량 ⚠️ 가장 시급

**현재 설정**
```

재진입 쿨다운: 없음
즉시 시장가 진입
```

**문제점**
- BARD: 신호가 1930원 → 체결 1997원 (슬리피지 3.5%) → 고점 매수
- 1차 손절 후 6초 만에 2차 신호 발생 → 즉시 재진입 → 연속 손절
- BB 상단 돌파 신호 발생 시점에 이미 급등 완료 상태

**개선 계획**
- [ ] **다음 봉 확인 후 진입** 로직 추가 (가장 중요)  
  → `strategyC.ts`에 `pendingEntryMarket: Map<string, number>` 추가  
  → 조건 충족 시 즉시 진입 X, 다음 틱(봉) 가격이 BB 상단 위에 있을 때 진입  
  → 급등 후 눌림 없이 그대로 올라갈 때만 진입
- [ ] **전략 C 전용 쿨다운 추가**  
  `config.ts`: `STRATEGY_C_COOLDOWN_MS = 30 * 60 * 1000` (30분)  
  `index.ts`: 전략 D, F와 동일한 쿨다운 맵 구조 적용
- [ ] 고변동성 구간 슬리피지 대응: 진입 전 5분봉 변동폭 체크 추가

### 전략 E — 박스권 BB수축+수평+하단터치+RSI40

**현재 설정**
```
BB_WIDTH_LOOKBACK = 100 (매우 엄격)
VOLUME_EXCLUDE_RATIO = 2 (거래량 2배 초과 시 제외)
```

**현황**
- 분석 기간 내 매수 신호 0회 → 조건이 너무 엄격하여 사실상 비활성 상태
- 박스권 + 저거래량 조건은 개념은 좋으나 실제 신호 발생 빈도 극히 낮음

**개선 계획**
- [ ] 단기 성과 관찰 중 → 현재는 `STRATEGY_E_ENABLED = true` 유지  
  → 신호 발생 시 별도 추적 필요
- [ ] 조건 완화 검토: `BB_WIDTH_LOOKBACK: 100 → 50`  
  → 최근 50봉 기준으로 완화하여 신호 발생 빈도 증가
- [ ] 또는 `STRATEGY_E_ENABLED = false`로 비활성화하고 집중 전략에 리소스 집중

### 전략 F — VWAP눌림목 + EMA21 + RSI40 + 양봉

**현재 설정**
```

ENTRY_BREACH_PCT = 0.7

```

**문제점**
- 쿨다운 5분으로 KAVA 당일 3회 연속 진입 (15:31, 16:43, 17:45)
- 진입이탈 손절 0.7%가 변동성에 비해 너무 좁음 → 조기 손절 남발
- KAVA 15:31: 신호가 100원 → 체결 101원 슬리피지로 즉시 손절

**개선 계획**
- [ ] `STRATEGY_F_COOLDOWN_MS`: `5분 → 60분`  
  → 당일 같은 종목 반복 재진입 방어
- [ ] `STRATEGY_F_ENTRY_BREACH_PCT`: `0.7 → 1.0`  
  → 진입이탈 손절 허용폭 확대 (변동성 흡수)
- [ ] `STRATEGY_F_VWAP_BUFFER_PCT`: `0.3 → 0.5`  
  → VWAP 붕괴 버퍼 확대 (일시 꼬리 달기 방어)
- [ ] `STRATEGY_F_TRAILING_ACTIVATE_PCT`: `1.0 → 0.7`  
  → 더 일찍 트레일링 활성화하여 수익 보존 기회 확대
- [ ] 체결가 기반 손절선 재산정 로직: 체결 후 `avgBuyPrice` 기준으로 손절가 재계산  
  → 슬리피지로 인한 즉시 손절 방지
